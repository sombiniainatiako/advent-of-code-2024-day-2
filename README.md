# Advent of Code Runner (Node.js)

This is a simple Node.js script that reads Advent of Code input from a file and runs the solution logic defined in `index.js`.

---

# 📦 Prerequisites

Before you start, make sure you have the following installed:

## Node.js

Download and install Node.js (LTS recommended):

https://nodejs.org/

After installation, verify:

```bash
node -v
```

# Input File Creation

1. Create a file in the project root called datasource.txt
2. Paste your input into the file

# Command Line Execution

Run the program using Node.js and pass the input file as argument

```bash
node index.js datasource.txt
```
